# clankie-cad

This repo is for sharing CAD models made with Clankie.

The goal is simple: make the STL files and OpenSCAD source files available so other people can print, modify, improve, and remix them. There will be many different objects in this repo over time, so individual model details may live alongside the files themselves rather than in this main README.

## What is included

This repository may contain:

- `.stl` files for 3D printing
- `.scad` files for OpenSCAD source models
- supporting notes, photos, or reference files when useful

Whenever possible, the editable OpenSCAD source is included so the designs can be adjusted instead of only reprinted as-is.

## Using the files

Download the STL if you just want to print an object.

Open the matching OpenSCAD file if you want to inspect, edit, resize, or regenerate the model.

Unless otherwise noted, dimensions are in millimeters.

## Remixing and sharing

If you modify any of the objects in this repo, please reshare your changes so other people can benefit too.

That is the main spirit of this repo: print, improve, remix, and share back.

## Notes

These models are shared as-is. Please check dimensions, fit, material choice, and print settings for your own use case before relying on a part.
